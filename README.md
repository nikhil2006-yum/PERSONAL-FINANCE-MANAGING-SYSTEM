# 💰 Personal Finance Manager

A simple command-line based **Personal Finance Manager** built using **Python** and **SQLite3**.  
This project helps users manage their income and expenses efficiently through a menu-driven interface.

---

## 🚀 Features

✅ Add Income and Expense Transactions  
✅ Store Data Permanently using SQLite Database  
✅ View All Transactions  
✅ Calculate Total Balance  
✅ Delete Transactions  
✅ Category-wise Expense Summary  
✅ Simple CLI Interface  
✅ Automatic Date & Time Recording  

---

## 🛠️ Technologies Used

- 🐍 Python 3
- 🗄️ SQLite3
- ⏰ Datetime Module
